Hot or Cold Friend Finder

The Hot or Cold Friend Finder is a web app game where users can send other users a location's coordinates. However instead of just showing the coordinates it gives the user a "Hot" or "Cold" notification based on if the distance since the last time they checked is smaller. In additional a meter is given that lets the user know a percent of how close they are with 0% being the distance they first checked from. The Git respository for this web app can be found at https://github.com/PhillipAC/findme
